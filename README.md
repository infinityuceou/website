Designed with a modern, minimal, and premium glass-themed UI system.

🌐 Live Website
---------------

🔗 [https://infinity2k25.in](https://infinity2k25.in)

🧠 About The Project
--------------------

INFINITY 2K26 is a national-level technical symposium featuring:

*   Multiple technical competitions
*   2-Day AI Workshop
*   Infinity Pass system (Coming Soon)
*   Interactive Gallery
*   Countdown to Event
*   Premium responsive design

The website follows a **glassmorphism + animated star-field** aesthetic to provide a clean and immersive experience.

🛠 Tech Stack
-------------

*   React (Vite)
*   Tailwind CSS
*   React Router
*   Framer Motion
*   React Three Fiber
*   Lucide Icons

🚀 Getting Started
------------------

### 1️⃣ Clone the Repository

    git clone https://github.com/infinityuceou/website.git
    cd website

### 2️⃣ Install Dependencies

    npm install

### 3️⃣ Run Development Server

    npm run dev

Open in browser:

    http://localhost:5173

⚡ Build for Production
----------------------

    npm run build

☁ Deployment
------------

Deployed using **Vercel**.

1.  Push project to GitHub
2.  Import into Vercel
3.  Select framework: Vite
4.  Deploy
